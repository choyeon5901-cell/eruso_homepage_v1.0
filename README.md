## 📌 프로젝트 개요

**(주)이루소**는 조치원 지역 최초로 **AI 드론 기반 스마트 약배송 시스템**을 도입하여 
첨단 인공지능 기술과 자율주행 드론을 활용한 안전하고 신속한 의약품 배송 서비스를 제공합니다.
